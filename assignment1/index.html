GITHUB OVERVIEW

Description: I will show you how to use github.
This is a simple and basic overview for newbies.

First of all you have to create an account on github(Click on navbar). If you already have one. Login to it.

You can also download github.

Now setup git on you system.
Open directory where you want to start you project.
You can open that directory from terminal(vs code) and then type git init

Create a new file index.html.

You can see changes you have made using git status
This will give you a small overview of which files are changed or created.

Then after creating/chaning a file. You have to add that file to the package. This paackage is just like a container. The container will contain all the new changes.
That will be send to the github repository. (Also check status here.)

Then after the files are added to container.
You have to add meaningful message to it.

Now you have to create a repository on you account to link with and upload you project there.

Enter details for your repository.

Then link your local project with repository.

To link enter command: git remote add origin -u #link#
where link is your repsitory address(in my case it was: https://github.com/haqnawaz99/MC-1.git)

Now you can push all your work to your repository by using comman git push -u origin main

You can also get the latest code from your project by using command git pull

WELL DONE!!! You have completed your tutorial
